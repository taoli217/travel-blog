# Maven Wrapper Scripts

This directory contains the Maven Wrapper scripts for the project.
The Maven Wrapper is a convenient way to provide a consistent build environment.

## Usage

Instead of using `mvn`, use:

- `./mvnw` (Linux/macOS)
- `mvnw.cmd` (Windows)

## First Time Setup

If this is the first time you're using the Maven Wrapper, you may need to run:

```bash
./mvnw -N
```

This will download the Maven Wrapper JAR and scripts.
